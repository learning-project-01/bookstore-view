# Media Controller
```
Action: Get all media.
curl -x GET http://localhost:8080/media
Action: Get media by id.
curl -x GET http://localhost:8080/media/1
Action: Create a media.
curl -x POST http://localhost:8080/media
Action: Update a media.
curl -x PUT http://localhost:8080/media/1
Action: Delete media by id.
curl -x DELETE http://localhost:8080/media/1
```
